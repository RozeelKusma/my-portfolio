# 🌑 Rojil Kusma — Portfolio

A modern, dark-themed developer portfolio built with **Next.js 14**, **TypeScript**, and **Tailwind CSS** — focused on clean design, smooth animations, and strong visual hierarchy.

## 🚀 Getting Started

```bash
pnpm install
pnpm dev
```

Open http://localhost:3000

---

## ⚙️ Customization

All editable content lives in:

```
app/data.ts
```

Update:

- Personal info
- Experience
- Projects
- Skills
- Contact details

---

## 🎨 Theme Customization

Colors are defined in:

```
tailwind.config.ts
```

Example:

```ts
colors: {
  ink: "#0A0A0F",
  paper: "#F4F1EA",
  accent: "#A78BFA",
  muted: "#9A9AA6",
  border: "#1C1C26",
  card: "#12121A",
}
```
